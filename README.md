# 💱 Conversor de Moedas

Aplicação web simples para conversão de moedas estrangeiras para Real Brasileiro (BRL), desenvolvida com **JavaScript puro**, HTML e CSS.

O projeto tem como objetivo praticar **manipulação de DOM**, **validação de dados**, **organização de código** e **boas práticas de front-end**, sem o uso de bibliotecas ou frameworks externos.

---

## 🚀 Funcionalidades

- Conversão de moedas:
  - Dólar Americano (USD)
  - Euro (EUR)
  - Libra Esterlina (GBP)
- Validação de entrada (apenas números)
- Formatação de valores no padrão brasileiro (R$)
- Interface limpa e responsiva
- Feedback visual do resultado
- Estrutura de código escalável

---

## 🛠️ Tecnologias utilizadas

- HTML5 (semântico)
- CSS3 (layout e estilização)
- JavaScript puro (ES6+)
